# FitCalc · 减脂计算器

> 科学减脂，无需节食 — 基于 Mifflin-St Jeor 公式的智能减脂方案生成器

## 🎯 功能

- **身体数据分析**：输入身高、体重、年龄、性别，实时显示 BMI 及分类
- **代谢计算**：BMR（基础代谢率）+ TDEE（每日总消耗）
- **热量缺口规划**：根据目标体重和时间计算每日所需热量，安全下限保护
- **三大宏营养素分配**：蛋白质 / 碳水 / 脂肪克数及比例
- **减脂时间线**：预测目标达成日期 + 进度条
- **每日三餐食谱**：早餐/午餐/晚餐，精确到克的食材配比
- **运动建议**：4种运动组合 + 卡路里消耗预估

## 🧮 核心公式

- **BMR (Mifflin-St Jeor)**
  - 男：`10 × 体重 + 6.25 × 身高 - 5 × 年龄 + 5`
  - 女：`10 × 体重 + 6.25 × 身高 - 5 × 年龄 - 161`
- **TDEE**：`BMR × 活动系数（1.2 ~ 1.9）`
- **热量缺口**：`1 kg 脂肪 ≈ 7,700 kcal`
- **蛋白质**：`1.8g × 体重(kg)`
- **运动消耗**：`MET × 3.5 × 体重(kg) ÷ 200` kcal/min

## 🚀 使用方式

1. 双击 `index.html` 在浏览器中打开
2. 输入身高、体重、年龄，选择性别和活动水平
3. 设定减重目标和计划周数
4. 点击"生成减脂方案"
5. 查看每日食谱和运动建议
6. 点击"换一份食谱"切换不同食材搭配

## 📝 关键设计决策

- **纯静态**：无需服务器，双击即用，可部署到 GitHub Pages
- **全局命名空间**：使用 `FIT.*` 而非 ESM，兼容 `file://` 协议
- **确定性食谱**：相同输入永远产出相同食谱（基于 dayIndex 种子）
- **安全底线**：每日摄入不低于 1200 kcal，缺口过大时红色警告
- **分量钳位**：所有食材克数限制在合理范围
