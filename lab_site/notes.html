学习笔记

❌ 常见错误 TOP 9

1. 用 def 定义类

def IceCreamStand(Restaurant): class IceCreamStand(Restaurant):

类用 class，函数用 def，不混用。

2. super.__init__ 缺少括号

super.__init__(restaurant_name, cuisine_type) super().__init__(restaurant_name, cuisine_type)

super 是类，super() 才是对象。必须先 () 再点方法。

3. *args 用 append 导致嵌套

self.flavors.append(args) → [("香草","草莓")] self.flavors.extend(args) → ["香草","草莓"]

*args 是元组，append 整个塞进去，extend 才逐个展开。

4. super() 传了未定义的变量

super().__init__(..., privileges, **kwargs) super().__init__(..., **kwargs)

变量必须先声明或作为参数传入才能使用。

5. 给父类传了过多位置参数

TypeError: 父类只收2个位置参数，你传了3个 额外信息用关键字参数传入 **kwargs

调用 super().__init__ 时只能传父类声明过的位置参数。

6. 循环自赋值 self.x = self.x

self.privileges = self.privileges → 右边不存在 self.privileges = self.kwargs.pop("privileges", [])

必须明确数据来源，不能自己赋值给自己。

7. dict.pop() 没给默认值

self.kwargs.pop("privileges") → KeyError self.kwargs.pop("privileges", []) → 安全

保底值：列表→[]，字典→{}，数字→0。

8. for 循环边遍历边修改

for num in nums: nums.remove(num) for num in nums[:]: nums.remove(num)

遍历副本，在原始列表上操作。

9. f-string 嵌套引号冲突

f"{dict["key"]}" → SyntaxError (3.11-) f"{dict['key']}" → 内外引号不同

Python 3.11 以前外层双引号内层必须单引号。

📋 方法速查表

方法作用改原列表？
append(x)加 1 个（整体）✅
extend(seq)加 N 个（拆开）✅
insert(i, x)指定位置插入✅
a + b合并（新列表）❌
a += b原地扩展（=extend）✅

🧠 super() 要点

  • super 是类，super() 是对象
  • 不用传 self，Python 自动处理
  • 只传父类声明过的位置参数
  • 额外参数走 **kwargs
